# Hello World Spring Boot

Minimal Spring Boot application that returns "Hello, World!" at `/`.

Quick start

1. Build:

```bash
cd /Users/priyanka/Documents/GitHub/helloworld-springboot
mvn -DskipTests package
```

2. Run:

```bash
mvn spring-boot:run
# or
java -jar target/helloworld-springboot-0.0.1-SNAPSHOT.jar
```

Then open `http://localhost:8080/`.
